# Rawda Interactive Quiz

موقع كويز تفاعلي ثابت يعمل على GitHub Pages بدون Backend أو SQLite.

## طريقة التشغيل
1. افتحي الفولدر في VS Code.
2. افتحي ملف `index.html`.
3. اضغطي كليك يمين ثم اختاري `Open with Live Server`.

## التعديلات الموجودة
- يطلب اسم الطالب قبل بداية الكويز.
- يختار 6 أسئلة عشوائية فقط من أصل 60 سؤالًا في كل محاولة.
- يسجل اسم الطالب والدرجة والنسبة والتاريخ داخل `localStorage` في نفس المتصفح.
- يمكن عرض سجل النتائج من الصفحة الرئيسية.
- يمكن تحميل النتائج كملف CSV.
- يمكن مسح سجل النتائج من نفس الجهاز.

## ملاحظة مهمة
التسجيل الحالي محلي على جهاز المتصفح فقط؛ لأنه لا يوجد Backend. عند رفع الموقع على GitHub Pages سيظل يعمل، لكن النتائج لن تتجمع مركزيًا من كل أجهزة الطلاب إلا إذا تم ربطه بخدمة خارجية مثل Google Sheets أو Firebase.
